# WebGIS — MVP

1. Dependencies:
   - frontend: npm install
   - functions: cd functions && npm install

2. Çalıştırma:
   - Frontend: npm run dev
   - Functions (lokal demo): npm run functions

3. Firebase:
   - firebase init (hosting, functions, firestore, auth)
   - firestore.rules dosyasını deploy edin
   - functions için gerçek service account ekleyin

Not: Bu repo iskelet olup, API anahtarlarını ve servis hesaplarını gerçek değerlerle güncelleyin.
